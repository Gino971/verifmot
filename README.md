# ODS9 Mobile

Mini-application autonome pour verifier des mots ODS9 et faire des recherches par motif.

## Lancer localement

Depuis le dossier parent :

python3 -m http.server 8000

Puis ouvrir :

http://127.0.0.1:8000/ods9-mobile/index.html

## Notes

- L'application charge ODS9 depuis ODS9.txt local.
- L'installation PWA est disponible via le navigateur si compatible.
